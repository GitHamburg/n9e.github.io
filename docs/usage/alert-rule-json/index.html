      告警规则-常用规则JSON

      本节整理一些常用的告警策略，这些告警策略都是导出的JSON文件，大家可以在告警规则管理页面，直接导入这些告警规则JSON，省去了自己的配置，具体操作路径：告警规则->选择要放置规则的业务组->右上角更多操作->导入告警规则->选择生效的集群->填入JSON即可。
